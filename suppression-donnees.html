DSCG Quiz - Demande de suppression des données

Dernière mise à jour : 10 mai 2026

Cette page permet aux utilisateurs de DSCG Quiz de demander la suppression de leur compte, de leur demande d'accès ou des données personnelles associées au service.

Comment faire une demande ?

Envoyez un email à l'adresse suivante :

dscgquiz@gmail.com

Vous pouvez utiliser le bouton ci-dessous pour préparer automatiquement l'email.

Envoyer une demande

Informations à indiquer

  • l'adresse email utilisée dans l'application ;
  • le nom et le prénom indiqués lors de la demande d'accès, si applicable ;
  • le type de demande : suppression du compte, suppression d'une demande d'accès, suppression de données de signalement ou autre demande liée aux données.

Données concernées

Selon la situation de l'utilisateur, la demande peut concerner les données suivantes : email, sexe, nom, prénom, tranche d'âge, ville ou région, UE souhaitées, demande d'accès, données d'authentification, signalements, commentaires, données techniques liées aux signalements, progression, scores, historique de quiz, tentatives, badges, favoris, préférences et statistiques de révision.

Données conservées localement

Certaines données peuvent être stockées uniquement sur le téléphone de l'utilisateur. Elles peuvent être supprimées en se déconnectant, en réinitialisant les données de l'application, en vidant le stockage de l'application depuis les paramètres Android ou en désinstallant l'application.

Délai de traitement

Les demandes sont traitées dans un délai raisonnable après réception. Une confirmation peut être envoyée lorsque la demande a été prise en compte ou si des informations complémentaires sont nécessaires pour identifier le compte concerné.

Contact

Pour toute question relative aux données personnelles : dscgquiz@gmail.com
